# MDGT база опытов

### Сервис для аутентификации протоколов лабораторных испытаний.

#### Функционал:
* база опытов
* хранение параметров и результатов как неструктурированных данных
* сервис доступа и формирования отчетов

#### Стек:
* postgresql
* fastapi
* sqlalchemy
